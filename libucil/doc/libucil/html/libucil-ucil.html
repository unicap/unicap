Convert colors between colorspaces. The colorspace field of dest needs to be set to the target colorspace, like in this example: 

src.colorspace = UCIL_COLORSPACE_RGB24;
src.rgb24.r = 0xff;
src.rgb24.g = 0xff;
src.rgb24.b = 0xff;
